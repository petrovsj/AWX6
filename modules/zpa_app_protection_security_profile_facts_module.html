zscaler.zpacloud.zpa_app_protection_security_profile_facts module – Retrieves App Protection Security Profile information.

Note

This module is part of the zscaler.zpacloud collection (version 1.0.0).

It is not included in ansible-core. To check whether it is installed, run ansible-galaxy collection list.

To install it, use: ansible-galaxy collection install zscaler.zpacloud. You need further requirements to be able to use this module, see Requirements for details.

To use it in a playbook, specify: zscaler.zpacloud.zpa_app_protection_security_profile_facts.

New in zscaler.zpacloud 1.0.0

  • Synopsis

  • Requirements

  • Parameters

  • Examples

Synopsis

  • This module will allow the retrieval of information about an App Protection Profile from the ZPA Cloud.

Requirements

The below requirements are needed on the host that executes this module.

  • Zscaler SDK Python can be obtained from PyPI https://pypi.org/project/zscaler-sdk-python/

Parameters

Parameter

Comments

id

string

The unique identifier of the AppProtection profile.

name

string

Name of the App Protection Security Profile.

provider

dictionary / required

added in zscaler.zpacloud 1.0.0

A dict object containing authentication details.

client_id

string / required

The unique API identifier for the API.

client_secret

string / required

Confidential key associated with the client ID.

cloud

string / required

The ZPA cloud provisioned for your organization

Choices:

  • "PRODUCTION"

  • "BETA"

  • "QA"

  • "QA2"

  • "GOV"

  • "GOVUS"

  • "PREVIEW"

customer_id

string / required

The unique identifier of the ZPA tenant.

Examples

- name: Get Details of All App Protection profiles
  zscaler.zpacloud.zpa_app_protection_security_profile_facts:
    provider: "{{ zpa_cloud }}"

- name: Get Details of a Specific App Protection profiles by Name
  zscaler.zpacloud.zpa_app_protection_security_profile_facts:
    provider: "{{ zpa_cloud }}"
    name: Example

- name: Get Details of a specific App Protection profiles by ID
  zscaler.zpacloud.zpa_app_protection_security_profile_facts:
    provider: "{{ zpa_cloud }}"
    id: "216196257331282583"
